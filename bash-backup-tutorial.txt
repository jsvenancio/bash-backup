## English

# bash-backup Tutorial

## 1. Configuration
1. Open `config.sh` using your preferred text editor:
   ```bash
   nano config.sh
   ```
2. Set the `SOURCE` and `DEST` variables for source and destination folders.

## 2. Running the Script
1. Open a terminal and navigate to the bash-backup directory:
   ```bash
   cd /path/to/bash-backup
   ```
2. Run the script:
   ```bash
   sh run.sh
   ```

## 3. Requirements
Ensure you have `rsync` and `mailutils` installed:
   ```bash
   sudo apt-get install rsync mailutils
   ```

## 4. Troubleshooting
- Check folder paths in `config.sh`.
- Ensure proper file permissions.
